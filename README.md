# moogia-ai-study-station
MoogIA 🐮🧠: Uma plataforma EdTech serverless e gamificada (estilo Duolingo). Transforma materiais em jornadas de estudo usando IA (Gemini). Inclui processamento in-browser de PDFs, geração automática de flashcards, mapas mentais, simulados e análise de editais. Uma SPA pronta para escalar e monetizar.
